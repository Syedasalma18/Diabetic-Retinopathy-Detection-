# Diabetic-Retinopathy-Detection-
# Diabetic Retinopathy Detection using Transfer Learning (ResNet-18)  This project implements an **AI-based system** to detect and classify **Diabetic Retinopathy (DR)** from retinal fundus images. DR is a severe complication of diabetes that can lead to blindness if not detected early, and manual screening is both time-consuming and error-prone.
